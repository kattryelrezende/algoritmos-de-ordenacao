Implementação de nove métodos de ordenação aplicados em vetores de inteiros de diferentes tamanhos.

PARA EXECUTAR O PROGRAMA, NO DIRETÓRIO DO PROJETO:
- Digite o comando 'make all'
- Em sequência, digite o comando 'make run'

O código foi estruturado a partir de uma interface amigável ao usuário, no qual você pode escolher exatamente a configuração que quiser.

Segue aqui o menu com os códigos de configuração presentes na execução do programa.

Para escolher seu algoritmo de ordenação, digite:
0: BubbleSort
1: SelectionSort
2: InsertionSort
3: ShellSort
4: QuickSort
5: HeapSort
6: MergeSort
7: Contagem de Menores
8: RadixSort

Para escolher o tamanho do vetor, digite:
0: 100 elementos
1: 1.000 elementos
2: 10.000 elementos
3: 100.000 elementos

Para escolher o formato do vetor, digite:
0: Aleatorios
1: Ordenados
2: Inversamente Ordenados

Caso seu vetor seja aleatório, para escolher qual dos 5 vetores aleatórios deseja-se ordenar, digite um número de 1 a 5.
